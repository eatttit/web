HTML이란 무엇인가?

Hypertet Markup Language (HTML) is the standard markup language for creting web pages and web aplications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML descrives the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
